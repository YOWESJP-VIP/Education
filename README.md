# TikTok Safe Link Landing Page

Landing page sederhana untuk digunakan sebagai link aman dalam iklan TikTok.

## Cara Pakai
1. Edit link tujuan pada file `index.html` (bagian `href="..."`).
2. Upload ke GitHub.
3. Aktifkan GitHub Pages:
   - Settings → Pages → Deploy from Branch → pilih `main` + `/root`.
4. Salin URL GitHub Pages sebagai link aman.

Contoh hasil:
https://username.github.io/tiktok-safe-link/
